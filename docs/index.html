TOUR

TOUR: Dynamic Topic and Sentiment Analysis of User Reviews for Assisting App Release

App reviews delivers user opinions and emerging issues (e.g., new bugs) about the app releases. TOUR provides entries for developers to analyzing users' opinions and detecting emerging issues from user reviews.

  Paper         Source Code

  Features

Topic-Sentiment Summary

Topic-based summarization of thousands of user reviews and further predicts sentiment polarities towards the topics.

Interactive Visualization

Visualize the features favorable and unfavorable by users through an interative and structured agenda.

User Opinion Identification

Identify user opinions towards app features based on customized polarity seed words;

Review Distillation

Distill actionable change tasks for improving app features.
